# Line Chart

## Example

````yaml
```chart
    type: line
    labels: [Monday,Tuesday,Wednesday,Thursday,Friday]
    series:
        - title: Title 1
        data: [1,2,3,4,5]
        - title: Title 2
        data: [5,4,3,2,1]
        - title: Title 3
        data: [8,2,5,-1,4]
```
````

![Line Chart Example Image](https://github.com/phibr0/obsidian-charts/raw/master/images/linechart.png)

## Line-Specific Modifiers

### `bestFit`

Add a line of best fit using linear regression. See [bestFit Modifiers](/Modifiers#bestfit-modifiers) for details.

### `stepped`

Render the line in a stepped fashion instead of a smooth diagonal. See [stepped Modifier](/Modifiers#stepped-modifier) for details.

### `fill`

Fill the area under the line. See [fill Modifier](/Modifiers#fill-modifier) for details.

### `tension`

Control the smoothness of the line. See [tension Modifier](/Modifiers#tension-modifier) for details.

### `spanGaps`

Connect data points across null values. See [spanGaps Modifier](/Modifiers#spangaps-modifier) for details.
